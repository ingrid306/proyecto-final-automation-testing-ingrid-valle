pre-entrega-automation-testing- Ingrid Valle 
Proposito del proyecto: realizar testeos de prueba automatizados en el sitio "https://www.saucedemo.com"



para ejecucion de test especificos utilizamos los comandos:

pytest tests\test_carrito_carga_productos.py pytest tests\test_catalogo.py pytest tests\test_login.py pytest tests\test_Navegar_a_pagina_carrito.py pytest tests\test_remover_producto_carrito.py


Tecnologías utilizadas

Lenguaje de programación: Python
Marco de pruebas: Pytest
Automatización UI: Selenium WebDriver
Gestión de Conductores: webdriver-manager
Pruebas de API: Solicitudes
Informes: pytest-html
Control de versiones: Git y GitHub


Instale todas las dependencias listadas en requirements.txt:
pip install -r requirements.txt
